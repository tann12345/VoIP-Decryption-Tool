# VoIP Decryption Tool

## Project Description
A powerful tool designed to decrypt VoIP communications enabling secure access and analysis of VoIP traffic. This project aims to provide researchers and network administrators with effective means to analyze encrypted VoIP calls.

## Features
- Decrypt various VoIP protocols
- Support for multiple encryption methods
- User-friendly command-line interface
- Generate detailed logs of decrypted sessions

## Installation Instructions
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/tann12345/VoIP-Decryption-Tool.git
   cd VoIP-Decryption-Tool
   ```
2. Install required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Ensure that you have the necessary configurations set up for your VoIP environments.

## Usage Guide
To start using the tool, run the following command:
```bash
python decryption_tool.py --input <input_file> --output <output_file>
```
Replace `<input_file>` with the path to the VoIP traffic you want to decrypt and `<output_file>` where the decrypted content will be saved.
